# drakon-renderer

## development environment

* start - `./start.sh`
* exit - `exit`

## hello world

* `cabal update`
* `cabal install --lib diagrams`
* `cabal install --lib diagrams-lib`
* `cabal install --lib diagrams-svg`
* `cabal install --lib base`
* `ghc HelloWorld.lhs` 
* `./HelloWorld -o hello-world.svg -w 400`
